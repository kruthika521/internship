Smart Shoe Rack

The Smart Shoe Rack is a hardware-based prototype developed using Arduino Uno. This project is designed to automatically organize and store shoes using IR sensors and a motorized mechanism. When a shoe is detected, the system activates a servo motor to place it in an available slot. The system improves space management and reduces manual effort in organizing footwear. Technologies Used: • Arduino Uno • IR Sensors • Servo Motors • Embedded C Programming This project demonstrates knowledge of embedded systems, hardware interfacing, and automation concepts.
